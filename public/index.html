About this tool

The Neurodivergent Perspective Translator takes any content and reframes it across 10 different cognitive styles — visual/spatial, pattern-seeking, hyperfocus, sensory/concrete, ADHD-friendly, rejection-sensitive, literal, big-picture, narrative, and question-based.

Built to support neurodivergent learners in accessing educational content in a way that works for their brain. No account required.

This tool uses the Claude API (Anthropic) to generate translations. Small API costs are covered by the project maintainer.

Content is not stored. Each translation is generated fresh and discarded.
