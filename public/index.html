# Neurodivergent Perspective Translator An accessible web tool that takes any content — a concept, question, lecture, or passage — and reframes it across 10 different cognitive styles tailored to neurodivergent minds. **No account required for users.** The site owner holds the API key server-side. --- ## The 10 Perspectives | Perspective | What it does | |---|---| | Visual / spatial | Metaphors, spatial relationships, mental imagery | | Pattern seeker | Underlying rules, systems, recurring structures | | Hyperfocus / deep dive | Maximum detail, nothing simplified | | Sensory / concrete | Real-world examples, tangible applications | | ADHD / attention variable | Bullet points, chunked, TL;DR at the top | | Rejection sensitive | Warm framing, encouraging tone, no judgment language | | Literal processor | No idioms, no implied meaning, everything explicit | | Big picture / systems | Why it matters, larger context, bigger significance | | Narrative | Story format, chronological, personified concepts | | Question-based | Reframed as a Q&A chain driven by curiosity | --- ## How it Works 1. User pastes any content into the site 2. The browser calls your **Netlify serverless function** (`/netlify/functions/translate`) 3. The function reads your API key from an **environment variable** (never exposed to users) 4. It calls the Claude API (Anthropic) and returns the translated text 5. All 10 perspectives generate in parallel --- ## Deploy to Netlify (Free Tier — ~10 min) ### Step 1: Get an Anthropic API key 1. Go to [console.anthropic.com](https://console.anthropic.com) 2. Create an account and add a small amount of credit (~$5 covers thousands of runs) 3. Go to **API Keys** → **Create Key** 4. Copy it — you'll need it in Step 4 ### Step 2: Put the code on GitHub 1. Create a free account at [github.com](https://github.com) if you don't have one 2. Click **New repository** → name it `nd-translator` → **Create repository** 3. Upload these files (drag & drop works): - `public/index.html` - `netlify/functions/translate.js` - `netlify.toml` - `package.json` - `.gitignore` Or use Git if you're comfortable: ```bash git init git add . git commit -m "Initial commit" git remote add origin https://github.com/YOUR_USERNAME/nd-translator.git git push -u origin main ``` ### Step 3: Connect to Netlify 1. Go to [netlify.com](https://netlify.com) → **Sign up for free** (can use your GitHub account) 2. Click **Add new site** → **Import an existing project** 3. Choose **GitHub** → authorize Netlify → select your `nd-translator` repo 4. Build settings (should auto-detect, but if not): - **Build command:** *(leave blank)* - **Publish directory:** `public` 5. Click **Deploy site** ### Step 4: Add your API key (the important part) 1. In Netlify, go to your site → **Site configuration** → **Environment variables** 2. Click **Add a variable** 3. Key: `ANTHROPIC_API_KEY` 4. Value: your API key starting with `sk-ant-...` 5. Click **Save** 6. Go to **Deploys** → **Trigger deploy** → **Deploy site** (to pick up the new env var) Your site is live! Netlify gives you a free URL like `nd-translator-abc123.netlify.app`. ### Step 5 (Optional): Set a custom domain In Netlify → **Domain management** → **Add a domain**. You can buy one (~$10/year) or use the free `.netlify.app` URL. --- ## Cost Estimate Each "Translate all 10" run makes 10 API calls to Claude Sonnet. | Usage | Estimated cost | |---|---| | Short input (~200 words) | ~$0.005 per run | | Medium input (~500 words) | ~$0.01–0.02 per run | | 100 runs/month | ~$0.50–2.00/month | | 1,000 runs/month | ~$5–20/month | Set a **usage limit** in your Anthropic console to cap spending. --- ## Rate Limiting / Abuse Protection The serverless function already: - Limits input to 8,000 characters - Returns clear error messages if the API key is missing or invalid For heavier shared use, consider adding: - A simple CAPTCHA (hCaptcha has a free tier) - Rate limiting via Netlify Edge Functions --- ## Local Development ```bash # Install Netlify CLI npm install # Create a local env file echo "ANTHROPIC_API_KEY=sk-ant-your-key-here" > .env # Run locally npm run dev # → Opens at http://localhost:8888 ``` --- ## Project Structure ``` nd-translator/ ├── public/ │ └── index.html ← The entire frontend (one file) ├── netlify/ │ └── functions/ │ └── translate.js ← Serverless API proxy ├── netlify.toml ← Netlify build config ├── package.json └── .gitignore ``` --- ## Customizing ### Change or add perspectives Edit the `PERSPECTIVES` array in `public/index.html`. Each perspective has: - `id` — unique string - `color` — dot color - `title` / `sub` — display labels - `prompt` — the instruction sent to Claude ### Swap the Claude model In `netlify/functions/translate.js`, change the `model` field: - `claude-sonnet-4-20250514` — best quality (default) - `claude-haiku-4-5-20251001` — faster, cheaper ### Update the About modal and footer Edit the relevant sections at the bottom of `public/index.html`. --- ## Potential Next Steps - Add a "Save my favorites" feature using localStorage - Export all 10 translations as a PDF - Add support for uploading a PDF or image directly - Grant funding: this tool has potential for educational equity and public health applications — see resources like the [NSF Convergence Accelerator](https://new.nsf.gov/convergence-accelerator) and [NIH SBIR/STTR](https://sbir.nih.gov/) --- ## License MIT — free to use, modify, and share.
